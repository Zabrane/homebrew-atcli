
# ATCLI

Cross-Platform Secure High-Speed Managed File Transfer Service

## Installation

Use the package manager [brew](https://brew.sh/) to install atcli.

```bash
brew tap alephtransfer/homebrew-atcli
brew install atcli
```

## Usage

```bash
atcli      --help
atcli up   --help
atcli down --help
```
