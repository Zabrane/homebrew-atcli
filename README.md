
# ATCLI

Cross-Platform Secure High-Speed Managed File Transfer Service

## Installation

Use [brew](https://brew.sh/) package manager to install atcli.

```bash
brew tap alephtransfer/homebrew-atcli
brew install atcli
```

## Usage

```bash
atcli      --help
atcli up   --help
atcli down --help
```
